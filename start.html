Как начать играть на ГусЛенд

Добро пожаловать на сервер ГусЛенд! Ниже ты найдёшь всё необходимое, чтобы начать своё приключение.

1. Версия Minecraft

Для подключения к серверу используй Minecraft Java Edition версии 1.20.1. Это стабильная версия, поддерживаемая сервером.

2. IP сервера

Подключись к нашему серверу, указав в клиенте Minecraft следующий IP адрес:

185.9.145.8:38937

3. Первый вход

После подключения ты появишься в стартовой зоне. Следуй инструкциям, чтобы выбрать направление и начать исследование мира.

4. Общие советы

  • Прочитай правила сервера.
  • Уважай других игроков — наш сервер дружелюбный и ламповый.
  • Не забудь заглянуть в раздел плагинов, чтобы понять механику сервера.
  • Если возникнут вопросы — обратись к модераторам или в Discord.

5. Discord

Для общения, новостей и обратной связи используй наш Discord-сервер (ссылка будет доступна позже).
